# Satistuffed

Go is training for go, let's go.

```
make init
```


# usage

```
satistuffed get 強化鉄板
 強化鉄板のレシピ1
 ├─ 鉄板のレシピ1 x 2 (余り: 10)
 │  ├─ 鉄板のレシピ1
 │  │  └─ 鉄インゴットのレシピ1
 │  │     └─ 鉄鉱石のレシピ1 (余り: 30)
 │  └─ 鉄板のレシピ1
 │     └─ 鉄インゴットのレシピ1
 │        └─ 鉄鉱石のレシピ1 (余り: 30)
 └─ ネジのレシピ1 x 2 (余り: 20)
    ├─ ネジのレシピ1
    │  └─ 鉄のロッドのレシピ1 (余り: 5)
    │     └─ 鉄インゴットのレシピ1 (余り: 15)
    │        └─ 鉄鉱石のレシピ1 (余り: 30)
    └─ ネジのレシピ1
       └─ 鉄のロッドのレシピ1 (余り: 5)
          └─ 鉄インゴットのレシピ1 (余り: 15)
             └─ 鉄鉱石のレシピ1 (余り: 30)

```

```
satistuffed get 強化鉄板 --origin
鉄鉱石: 120
```

```
satistuffed get ヘビー・モジュラー・フレーム --origin
鉄鉱石: 1350
石炭: 450
石灰岩: 180
```
